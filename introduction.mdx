## Welcome to OrionRides

OrionRides is a full-stack car rental web application that allows users to explore available vehicles, book cars, and manage rentals seamlessly. Built with the **MERN stack** (MongoDB, Express, React, Node.js), OrionRides provides a modern and scalable solution for car rental services.

## Key Features

<AccordionGroup>
  <Accordion title="User Authentication" icon="lock">
    Secure user signup and login with JWT-based authentication. Passwords are hashed using bcrypt, and authentication tokens are stored in HTTP-only cookies for enhanced security.
  </Accordion>

  <Accordion title="Car Listings" icon="car">
    Browse a comprehensive catalog of vehicles available for rent. Each vehicle includes detailed information, images, pricing, and availability status.
  </Accordion>

  <Accordion title="Booking System" icon="calendar-check">
    Rent cars for specified dates with real-time price calculations. The system validates availability and prevents double bookings automatically.
  </Accordion>

  <Accordion title="User Dashboard" icon="user">
    Manage your rentals, view rental history, and track active bookings. Users can cancel reservations and update their profile information.
  </Accordion>

  <Accordion title="Responsive Design" icon="mobile">
    Fully optimized for all device sizes with smooth animations powered by Framer Motion. The interface adapts seamlessly from mobile to desktop.
  </Accordion>
</AccordionGroup>

## Technology Stack

OrionRides is built with modern web technologies:

<CardGroup cols={2}>
  <Card title="Frontend" icon="react">
    - **React 18** - UI library
    - **Vite** - Build tool and dev server
    - **React Router** - Client-side routing
    - **Framer Motion** - Animations
  </Card>
  <Card title="Backend" icon="server">
    - **Node.js** - JavaScript runtime
    - **Express** - Web framework
    - **MongoDB** - Database
    - **Mongoose** - ODM for MongoDB
  </Card>
  <Card title="Authentication" icon="shield">
    - **JWT** - Token-based auth
    - **bcrypt** - Password hashing
    - **cookie-parser** - Cookie handling
  </Card>
  <Card title="DevOps" icon="cloud">
    - **Vercel** - Deployment platform
    - **Git** - Version control
    - **ESLint** - Code linting
  </Card>
</CardGroup>
